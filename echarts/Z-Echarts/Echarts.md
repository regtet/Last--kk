# 相关配置

+ title: 标题组件
+ tooltip:提示框组件
+ legend:图例组件
+ toolbox：工具栏
+ grid：直角坐标系内绘图网格
+ xAxis：直角坐标系grid中的x轴
+ yAxis：直角坐标系grid中的y轴
+ series：系列列表。每个系列通过type决定自己的图表类型（什么类型的图标）
+ color：调色盘颜色列表
